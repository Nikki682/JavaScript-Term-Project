*Main Menu*
There are four game options to choose from. Click on the link in the box of your chosen game.

Game 1:Jigsaw
This easy-to-play game is for all ages and abilities. The object is to click on the pieces at 
the bottom of the screen and drag them over the guide picture and drop them to make the completed picture. 
Future updates include: a timer, reset button, multiple images, and a completion event. 
language: html, css, JavaScript, JQuery

Game 2: Escape
This side-scrolling canvas game is for all ages and abilities. The object in this game is to move 
the lion through the holes in the maze of oncoming cage bars. To do this you use the four directional arrows on your keyboard.
Future updates include: a restart button, high scores, and alternative images. 
language: html, css, JavaScript

Game 3: Shoot
This fun canvas shooter is for all ages and abilities. The object of the game is to click on(shoot) 
all the birds before they reach the left side of the screen.
Future updates include: a restart button, parallax background, new sprite image, and background music.
language: html, css, JavaScript

Game 4: Peanut Smash
This whack-a-mole type game is for all ages and abilities. The object is to click on(smash) the peanuts 
as they pop out of the bags. They are quick.
Future updates include: sound effects, top scores, and new images.
language: html, css, JavaScript
